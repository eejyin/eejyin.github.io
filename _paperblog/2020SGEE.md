## Summary

This work proposes a reliability assessment method for distribution networks that explicitly
accounts for the **differentiated reliability demand of end users**. Rather than treating every
load point identically, the method weights reliability indices by how sensitive each end user is
to interruptions, so that the assessment better reflects the service quality that customers
actually experience.
