**Role:** Topic Investigator &nbsp;&middot;&nbsp; **Timeline:** Jan. 2020 – Dec. 2021

Funded by the State Grid Corporation Headquarters Technology Project (No. 5400-202012118A-0-0-00).

- Proposed a distribution-network reliability evaluation index based on value-engineering theory.
- Established a comprehensive reliability evaluation index system considering the number of users and power-supply capacity.
- Computed reliability index weights using the Analytic Hierarchy Process (AHP)–Entropy Weight Method (EWM).
